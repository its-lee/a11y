# a11y

Vue3 + TS repository containing an a11y playground, hosted [here](https://its-lee.github.io/a11y/#/).

## Project setup

```
npm ci
```

### Compiles and hot-reloads for development

```
npm run dev
```

### Compiles and minifies for production

```
npm run build
```

### Run your unit tests

```
npm run test:unit
```

### Lints files

```
npm run lint
```
